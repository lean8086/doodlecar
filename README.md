# Doodle Car
This is a simple experiment with some physics laws like acceleration and inertia.

[Try demo!](http://llinares.github.com/doodlecar/)

## Instructions
* ↑ or W: Throttle
* ↓ or S: Brake
* ← or A: Turn left
* → or D: Turn right

## Technologies
* 2D canvas
* Request Animation Frame
* Multiple CSS3 background patterns
