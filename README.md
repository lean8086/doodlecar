# Doodle Car
A simple experiment with some physics laws like acceleration and inertia.

[Try demo!](http://llinares.github.com/doodlecar/)

## Instructions
* ↑ or W: Throttle
* ↓ or S: Brake
* ← or A: Turn left
* → or D: Turn right
